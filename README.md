# DuoRide

## Functionality

**DuoRide** is a device/app that is supposed to keep you company by using voice lines. 

**DuoRide** can:
* Greet
* say good bye
* stream you music
* be humourus 

## Must haves
 
**Soft:** 
- python
  - nfcpy
  - mpg123

**Hard:** 

* Raspberry Pi (3b+)
* nfc reader
