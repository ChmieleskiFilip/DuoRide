# DuoRide

## Functionality

**DuoRide** is a device/app that is supposed to keep you company by using voice lines. 

**Roadmap** 🗺️
- [x] Greet
- [ ] Say good bye
- [ ] Stream you music
- [ ] Be humorous 
- [ ] Remind you if you forget you phone
- [ ] Reports: trafic, cameras and even if you go above the speed limit.

## Built with 🏗️
**Soft:** 
- python
  - nfcpy
  - mpg123

**Hard:** 
* Raspberry Pi (3b+)
* nfc reader
